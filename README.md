# Broadcast Server - Plataforma de Datos (PL/SQL Only)

Este proyecto define la arquitectura de datos del sistema **Broadcast Server**, basado en el enunciado del trabajo final:

- **Bloques anónimos de PL/SQL**
- **Creación de esquema completa (DDL)**
- **Población de datos semilla (DML)**
- **Casos de uso críticos**
- **Control transaccional explícito**
- **Manejo de excepciones**
- **Validaciones in-block**
- **Patrón table-driven**
- **Separación de comandos y consultas**

Este repositorio se ejecuta en **Oracle XE dentro de Codespaces**, permitiendo reproducir el entorno sin instalación local.
